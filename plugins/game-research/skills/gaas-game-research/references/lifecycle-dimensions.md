# 生命周期维度定义 v1.0

> **用途：** 为单个资料片/大版本的生命周期研究生成「版本级画像」，追踪从上线到衰退的完整过程。
> **版本：** v1.0

---

## 使用说明

- 本维度表在现有三层框架之上叠加生命周期视角，聚焦「某个版本期间」的体验与结构。
- 适用于：资料片、大版本更新（如 X.0）、赛季章节（如 Episode 1-3）等有时间边界的运营单元。
- 生命周期评分需与游戏整体评分区分——同一游戏的不同版本可能在生命周期维度上差异巨大。

---

## L1 生命周期阶段划分

> 描述玩家活跃/留存随时间的变化模式，以及版本的有效寿命。

| 维度 | 值类型 | 说明 |
|------|--------|------|
| **L1a 活跃曲线形态** | 离散四档 | 玩家活跃/留存随时间的变化模式 |
| **L1b 生命周期长度** | 连续谱 1-10 | 1=版本内容1-2周内被消耗完 → 10=版本持续产生意义超过1年 |

### L1a 活跃曲线形态 档位说明

| 档位 | 描述 | 典型案例 |
|------|------|---------|
| **高开低走** | 上线峰值极高，随后快速衰退 | 原神大版本（内容消耗快，长草期长） |
| **渐进稳定** | 上线平稳，逐步积累到稳定 plateau | FF14资料片（MSQ驱动，补丁节奏稳定） |
| **脉冲式** | 随每次内容更新出现活跃峰值 | 魔兽世界赛季制 |
| **持续衰退** | 从上线起就持续流失，无明显反弹 | 命运2 Episodes |

## L2 内容消耗与供给

> 描述版本内容被玩家消耗的速度，以及版本后期的内容供给强度。

| 维度 | 值类型 | 低端（1）| 高端（10）|
|------|--------|---------|---------|
| **L2a 内容消耗周期** | 连续谱 | 内容在1-2周内被消耗完 | 内容可持续消耗3个月以上 |
| **L2b 长草期密度** | 连续谱 | 几乎无长草期，一直有内容 | 长草期极长，依赖日常/循环维持 |
| **L2c 终局内容强度** | 连续谱 | 版本后期无值得重复的内容 | 版本后期有高度可重复的终局玩法 |

## L3 驱动力结构

> 描述不同阶段的核心玩家驱动力。一个版本的生命周期中，驱动力通常会变迁。

| 维度 | 值类型 | 说明 |
|------|--------|------|
| **L3 驱动力变迁** | 离散多选（分阶段）| 不同阶段的核心玩家驱动力 |

### L3 驱动力类型

| 选项 | 描述 |
|------|------|
| 叙事驱动 | 主线/支线剧情是玩家上线的核心理由 |
| 探索驱动 | 新地图/新区域探索是核心动力 |
| 角色/收藏驱动 | 新角色/新皮肤/新装备获取是核心动力 |
| 社交驱动 | 与他人一起游玩（组队、公会、PVP）是核心动力 |
| 数值驱动 | 数值成长/装备提升是核心动力 |
| 成就驱动 | 完成挑战/收集/100%是核心动力 |
| 日常惯性 | 每日任务/签到形成的习惯性登录 |

要求：为版本划分3-4个阶段（如「上线期」「稳定期」「衰退期」「终局期」），标注每个阶段的主导驱动力。

## L4 版本遗产

> 描述版本结束后，其内容以什么形态存在，以及对后续版本的影响。

| 维度 | 值类型 | 说明 |
|------|--------|------|
| **L4a 内容留存形态** | 离散多选 | 版本结束后，该版本的内容以什么形式存在 |
| **L4b 对后续版本影响** | 连续谱 1-10 | 1=无影响，完全被遗忘 → 10=根本性改变游戏结构 |

### L4a 内容留存形态 选项

| 选项 | 描述 |
|------|------|
| 永久常驻 | 所有内容进入常驻库，新玩家随时可体验 |
| 部分常驻 | 核心内容常驻，限时活动消失 |
| 仅奖励常驻 | 内容消失，但通过复刻/兑换保留奖励获取途径 |
| 纯历史 | 内容完全消失，仅存在于玩家记忆和Wiki中 |
| 系统遗产 | 内容消失，但引入的系统/机制被后续版本继承 |
