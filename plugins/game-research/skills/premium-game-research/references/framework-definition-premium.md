# 游戏属性维度表——框架定义（买断制版）

> **用途：** 通用研究工具。给买断制游戏生成「维度画像」，用于描述和横向比较不同游戏的体验特征与产品结构。
> **版本：** v1.0-premium
> **适用对象：** 买断制（Premium / Buy-to-Play）游戏，优先覆盖开放世界与宽线性 3A/准3A 3D 游戏。

---

## 使用说明

- 本表为**描述工具**，不是问卷量表。目标是给游戏打出一份客观的属性画像。
- 内容逻辑优先于产品逻辑：体验气质层描述游戏「是什么」，应最先确定；产品结构层描述「怎么组织和交付体验」，在体验逻辑确立后填写。
- 付费结构（买断/DLC/豪华版等）和可达性设计（难度选项/辅助模式等）作为补充信息写在报告概况中，不作为正式评分维度。

---

## 第一层：体验气质层

> 描述玩家感受到什么。这一层是游戏核心竞争力的所在，也是最难更改的设计决策。

| 维度 | 值类型 | 低端（1）| 高端（10）|
|------|--------|---------|---------|
| **A 叙事控制权** | 连续谱 1-10 | 内容完全由设计者编写和编排 | 内容完全由玩家与系统规则交互涌现 |
| **B 自我代入深度** | 连续谱 1-10 | 玩家与角色无认同感，纯旁观者 | 玩家完全将自身投射到玩家角色 |
| **C 伙伴/NPC依恋密度** | 连续谱 1-10 | NPC为纯功能性存在 | 玩家对特定角色有强烈真实情感依恋 |
| **D 世界自洽感** | 连续谱 1-10 | 世界是纯背景板，不需要理解 | 世界有独立运行逻辑，理解世界本身是乐趣 |
| **E 题材辨识度** | 连续谱 1-10 | 题材通用，与同类游戏难以区分 | 题材高度独特，美学身份一眼可识别，无可替代 |
| **F 玩家能动性** | 连续谱 1-10 | 玩家完全接收官方内容 | 玩家是最主要的内容来源 |
| **Q4 情感消费节奏** | 离散三档（可组合）| — | — |

### Q4 情感消费节奏 档位说明

| 档位 | 描述 | 典型案例 |
|------|------|---------|
| **消耗型** | 核心情感体验是一次性事件，内容被「用完」后体验结束 | RDR2主线、赛博朋克2077、最后生还者系列、底特律变人 |
| **沉浸型** | 情感体验随时间积累加深，旧内容持续产生新意义 | 动物森友会、星露谷物语、旅途 |
| **循环型** | 体验设计为可重复，每次循环提供近似的情感弧线 | 黑魂系列NG+、杀戮尖塔roguelike循环、文明6 |

> 一款游戏可同时具备多种节奏类型，填写时注明主次即可。

---

## 第二层：叙事架构层

> 描述内容是如何组织和交付的。这一层是体验气质的工业实现方式。

| 维度 | 值类型 | 说明 |
|------|--------|------|
| **N1a-P 叙事节奏结构** | 连续谱 1-10 | 1=叙事均匀分布，全程保持稳定的叙事密度<br>10=叙事高度集中在特定节点，大量游玩时间为非叙事体验 |
| **N1b 叙事交付方式** | 连续谱 1-10 | 1=完全推送型（过场/对话框主动呈现）<br>10=完全潜伏型（嵌入世界，玩家主动探索触发）|

### N1a-P 叙事节奏结构 参考

| 位置 | 描述 | 典型案例 |
|------|------|---------|
| 低端（1-3） | 全程叙事密度高且均匀，玩家始终处于叙事驱动的体验中 | 最后生还者系列、Alan Wake 2、战神：诸神黄昏 |
| 中间（4-6） | 叙事与非叙事体验交替，存在明显的叙事密集段和探索/战斗段 | 巫师3、赛博朋克2077、对马岛之魂 |
| 高端（7-10） | 大量游玩时间为非叙事体验（战斗、探索、系统交互），叙事集中在少数关键节点 | 艾尔登法环、塞尔达：王国之泪 |

## 第三层：产品结构层

> 描述游戏作为产品如何组织和交付其完整体验。买断制游戏没有持续运营，但有自己的结构性设计选择。聚焦 Core Game Loop 和整体游戏体验。

---

### P1 叙事分支密度

*玩家的选择在多大程度上改变叙事走向。*

| 值类型 | 低端（1）| 高端（10）|
|--------|---------|---------|
| 连续谱 1-10 | 完全线性，无分支，所有玩家体验相同的故事 | 高度分支，玩家选择实质性改变故事走向和结局 |

---

### P2 完成弧线

*游戏如何定义"结束"，以及结束后还剩什么。*

**值类型：** 各选项独立打权重 1-10，反映该选项在游戏中的实际比重。

| 选项 | 描述 |
|------|------|
| 单次通关 | 游戏有明确终点，通关后自然结束 |
| 多结局/NG+ | 设计上鼓励多周目，每次提供不同体验 |
| 开放沙盒 | 无明确终点，玩家自定义目标 |
| 通关后内容 | 通关后开放额外内容（隐藏BOSS、后日谈、挑战模式等）|

---

### P3 内容组织结构

*游戏内容的空间和流程组织方式。*

**值类型：** 离散选择——选择一个主导类型，可标注次要类型。

| 选项 | 描述 |
|------|------|
| **一体化开放世界** | 无缝连通的大型世界，玩家自由移动（GTA V、艾尔登法环） |
| **区域制开放** | 多个独立开放区域，区域间有加载或过渡（巫师3、地平线） |
| **章节制** | 内容按章节/关卡线性推进，每章内可能有一定自由度（Alan Wake 2） |
| **Hub-Spoke** | 中心区域连接多个任务/关卡区域（Control、战神） |

---

### P4 核心循环

*玩家在分钟/小时级别的主要活动模式。*

**值类型：** 各选项独立打权重 1-10，反映该循环在游戏中的实际比重。

| 选项 | 描述 |
|------|------|
| 战斗循环 | 遭遇→战斗→奖励→准备→下一次遭遇 |
| 探索-发现循环 | 移动→发现兴趣点→探索→获取信息/物品→继续移动 |
| 叙事推进循环 | 接受任务→体验剧情→做出选择→推进故事 |
| 系统交互-涌现循环 | 实验→观察系统反应→学习规则→创造性运用 |
| 收集-定制循环 | 发现/获取外观元素（装备、载具、家居）→搭配/展示→寻找下一个目标 |
| 建造-创造循环 | 收集资源→设计→从零组装/建造→评估→迭代 |

---

### P5 支线与主线关系

*支线内容与主线的结构关系。*

**值类型：** 各选项独立打权重 1-10，反映该关系类型在游戏中的实际比重。

| 选项 | 描述 |
|------|------|
| 支线独立无关联 | 支线任务自成一体，与主线叙事和进度无交叉 |
| 支线提供世界补充 | 支线丰富世界观和角色理解，但不改变主线走向 |
| 支线结构性影响主线 | 支线完成情况实质性影响主线走向、可用资源或结局 |

---

### 成长/进度结构

*玩家随时间积累的是什么。*

**值类型：** 各选项独立打权重 1-10，反映该选项在游戏中的实际比重。

| 选项 | 描述 |
|------|------|
| 数值成长 | 角色/装备数值持续提升，直接影响战斗能力 |
| 收藏扩展 | 积累更多选项（角色、皮肤、道具），不改变基础强度 |
| 资产积累 | 积累游戏内财富、地产、影响力 |
| 理解加深 | 通过游玩加深对世界/系统/机制的理解 |
